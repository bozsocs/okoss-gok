
JEGYZET – Ágazati Vizsga – P1: Alap Python függvények

Alapműveletek:
- Összeadás, kivonás, maradék képzés, abszolút érték számítása.
- Saját függvények írása +, -, % műveletekkel.
- Feltételes elágazásokkal (if-else) kell kiválasztani a kisebb vagy nagyobb számot.

Oszthatóság, párosság:
- A szám osztható-e adott számmal? Ez % operátorral vizsgálható.
- Ha szam % 3 == 0, akkor osztható hárommal.
- Hasonlóan működik a 2, 4, 7, stb. esetén is.
- Páros szám: osztható kettővel, azaz szam % 2 == 0.

Geometria:
- Kocka térfogata: oldal^3
- Téglatest térfogata: a * b * c
- Négyzet kerülete: 4 * oldal
- Négyzet területe: oldal^2
- Négyzet átlója: √2 * oldal
- Téglalap átlója: √(a^2 + b^2)
- Kör területe: π * r^2
- Kör kerülete: 2 * π * r
- Derékszögű háromszög átfogója: √(a^2 + b^2)
- Derékszögű háromszög területe: a * b / 2

P2 – Lista, karakter, szöveg műveletek

Lista műveletek:
- Listák bejárása for ciklussal.
- Lista elemeinek összeadása: változóban gyűjtöd.
- Átlag: összeg / darabszám
- Legkisebb/legnagyobb: végig hasonlítással.
- Elemszám: megszámolod feltétel szerint (pl. negatív, páros).
- Kiválogatás: új listába gyűjtöd a megfelelőt.
- Keresés: ciklus, if, visszatérés.
- Első előfordulás: index-szel követed.

String műveletek:
- Karakterek elérése indexeléssel (s[0], s[-1])
- Keresés: végignézés karakterenként
- Tartalmaz-e valamit: "lorem" in szoveg

Figyelj:
- Üres lista vagy string → térj vissza 0 vagy None értékkel.
- Ne használj tiltott beépített függvényeket!
